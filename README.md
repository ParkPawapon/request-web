# request-web

Production frontend foundation for the request platform.

## Stack

- Bun
- Next.js App Router
- React
- TypeScript strict mode
- MUI
- Tailwind CSS
- Zod
- Vitest and React Testing Library

## Commands

```bash
bun install
bun run typecheck
bun run lint
bun run test
bun run build
```

## Scope

Legacy UI and business logic have not been migrated yet. The current codebase
establishes architecture boundaries, environment validation, API client
normalization, theme tokens, route states, and test scaffolding.
